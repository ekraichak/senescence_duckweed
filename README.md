# Demographic senescence of _Wolffia brasiliensis_ and _Lemna aequinoctiales_

The datasets and codes here are from the publication "Demographic senescence and lifespan extension by caloric restriction in two duckweed species (_Wolffia brasiliensis_ and _Lemna aequinoctialis_)" in _Aquatic Botany_. The R code and dataset for each species is in their own folder.
